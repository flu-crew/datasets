Outputs from the Flumina pipeline run on the samples:

1) curated_database.csv: the database of selected functional variants of interest

2) raw-variant-table.csv: the raw VCF file results converted to a table and aggregated for all the samples

3) all_sample_amino_acids.txt:  the raw-variant-table translated to amino acids

4) curated_amino_acids.txt: the all_sample_amino_acids.txt matched to the curated_database.csv
